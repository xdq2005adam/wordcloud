# 实时词云网站

一个实时词云展示系统，支持通过手机扫码输入内容，实时显示在词云中。

## 功能特点

- 📱 左侧显示二维码，右侧展示词云
- 🎨 词云横版排放，彩色文字，大小不一
- 🔄 实时更新，无需刷新页面
- 📲 移动端友好的输入界面

## 安装步骤

1. 安装依赖：
```bash
npm install
```

2. 启动服务器：
```bash
npm start
```

或者使用开发模式（自动重启）：
```bash
npm run dev
```

3. 打开浏览器访问：
   - 主页面：http://localhost:3000
   - 输入页面：http://localhost:3000/input.html

## 使用说明

1. 在电脑上打开主页面（http://localhost:3000）
2. 使用手机或平板扫描左侧的二维码
3. 在手机上输入内容并点击发送
4. 内容会实时显示在电脑端的词云中

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **后端**: Node.js, Express
- **实时通信**: Socket.IO
- **词云库**: WordCloud2.js
- **二维码**: QRCode.js

## 注意事项

- 确保电脑和手机在同一网络下
- 如果无法访问，请检查防火墙设置
- 服务器默认运行在 3000 端口，可在 `server.js` 中修改
